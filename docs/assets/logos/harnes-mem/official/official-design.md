# Harness-mem Official Design

## Canonical assets

- Logo: `harness-mem-logo-official.jpg`
- Mascot: `harness-mem-mascot-official.jpg`

## Usage rules

1. Keep original aspect ratio.
2. Keep brand color contrast (orange on dark background) when possible.
3. Do not stretch, rotate, or recolor the logo mark.
4. Do not alter mascot facial expression or chest motif for official channels.
5. Do not add extra text over the logo or mascot.

## Recommended usage

- Use the logo for README headers, npm/GitHub visuals, and release graphics.
- Use the mascot for community-facing sections and friendly brand moments.
- Keep mascot usage separate from technical feature explanations.

## Source selection

- Logo selected from nano-banana variant 05.
- Mascot selected from refined mascot variant 02.
